# 🦷 Dental Clinic Website

A modern, responsive, and user-friendly Dental Clinic Website built with React.js. This project demonstrates a clean user interface, smooth user experience, and a professional design suitable for dental clinics and healthcare businesses.

## ✨ Features

* Modern & Responsive Design
* Hero Section
* About Clinic
* Services Section
* Doctor Information
* Patient Testimonials
* Appointment Section
* Contact Form
* Google Maps Integration
* Smooth Animations

## 🛠 Tech Stack

* React.js
* Tailwind CSS
* JavaScript
* Framer Motion
* Lucide React

* Home Page
* Services
* Doctors
* Appointment Section
* Contact Section
